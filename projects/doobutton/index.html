DooButton

DooButton is a big Red Internet connected button that will ping a web server if you press it. It will toggle on or off based on its previous state. It behaves like a “Call attendant” button in an airplane. The web server it connects to can be customised.

The DooButton can be used in museums, restaurants and other interactive installations.

The DooButton is based on the ESP8266. The technical details are in the PDF below.
